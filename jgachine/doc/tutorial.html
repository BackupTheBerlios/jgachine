JGachine tutorial

JGachine is a networked game engine for 2D multi-player (networked) games. JGachine is written in Java and C++. The games themselves are written in pure Java and can be loaded via network. (JGachine as game browser)

Getting a java development kit

I assume you are familar with Java or are willing to get started (other tutorials). There are many choices for java development. Here is a mall excerpt (with focus on developing games for the JGachine):
  • Get GNU gcc/gcj java for your platform (recommended since JGachine uses it and it is free software)
  • If you are on windows mingw might be a good choice
  • Sun's Java 2 Platform, Standard Edition (J2SE) SDK
  • ...
Write a hello world program and test it.

Getting the JGachine and its development files

Download JGachine for your platform.

Setup CLASSPATH

Your CLASSPATH must include jgachine.jar (and perhaps vecmath.jar if your java implementation does not provide javax.vecmath)

Hello JGachine!

Jgachine games are similar to java applets. [todo: ...]. We must write at least one class which implements the Runnable interface which means the class must have a 
void run()
method. The Hello.java: 
import org.jgachine.*;
import org.jgachine.scenegraph.*;

//! all JGachine games must implement the Runnable interface
class Hello implements Runnable
{
    public void run(){
	// create scenegraph
	Node root = (Node)new Scale(new Vector2f(64,64)).addNode(new Text("Hello JGachine!"));
	
	// main game loop
	while (!JGachine.quit) {
	    // get input
	    JGachine.poll();
	    // clear screen
	    JGachine.clear();
	    // paint scenegraph
	    root.apply();
	    // swap front and back buffer
	    JGachine.swapBuffers();
	}
    }
}
Now let us compile the class: 
javac Hello.java
Now let's run the game: 
jgachine Hello
You should see "Hello JGachine!" in the lower left corner.

Advanced examples are in the tutorial directory: pong1,pong2
